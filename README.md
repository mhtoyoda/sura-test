# Teste Toyoda - Sura

Api de Teste Sura

## Executar o comando no Terminal:
 ```
 docker-compose up
 ```
 
### Prerequisites

- Java 8
- Maven 3.6.0
- Docker
- Docker Compose

### Autenticação
```
http://localhost:8080/api/login

{
    "username": "email",
    "password": "senha"
}
```
** Obter Response Header Authorization

### URL Contexto

```
http://localhost:8080/api/
```


### SWAGGER URL
```
http://localhost:8080/swagger-ui.html
```
